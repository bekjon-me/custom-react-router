### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/bekjon-me/custom-react-router.git
```

-

### Install

Install packages:

```sh
  cd custom-react-router
  yarn
```

or

```sh
  cd custom-react-router
  npm install
```

-

### Usage

To run the project, execute the following command:

```sh
  yarn start
```

or

```sh
  npm start
```
